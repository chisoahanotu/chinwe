Not a clinical decision tool
Chinwe is an AI educational aid. It will make mistakes. Always verify with your attending and primary sources.
🔒
De-identify before you type
Chinwe is not HIPAA-compliant. Strip these before entering anything:
Name → "the patient"
MRN / IDs → omit
Date of birth → age only
Specific dates → "3 days ago"
Address → omit
Phone / email → omit
✓ Age, sex, vitals, labs, exam, PMH, meds — all fine.
📋
Your responsibility
By using Chinwe you accept full responsibility for appropriate de-identification per HIPAA Safe Harbor. When in doubt, leave it out.
